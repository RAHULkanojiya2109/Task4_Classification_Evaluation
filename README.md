# AI & ML Task 4

## Classification Models, Evaluation Metrics & Handling Imbalanced Data

### Objective
This project demonstrates machine learning classification techniques using the Breast Cancer Wisconsin Dataset. It focuses on model evaluation, confusion matrix analysis, ROC-AUC evaluation, and handling imbalanced datasets.

## Dataset
Breast Cancer Wisconsin Dataset (scikit-learn)

## Models Used
- Logistic Regression
- Balanced Logistic Regression
- Decision Tree Classifier

## Concepts Covered
- Classification
- Feature Scaling
- Train-Test Split
- Confusion Matrix
- Precision
- Recall
- F1 Score
- ROC Curve
- AUC Score
- Handling Imbalanced Data
- Model Comparison
